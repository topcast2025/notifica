Tutorial de Instalação

Guia completo para instalar e configurar o módulo WhatsApp no seu WHMCS

Introdução

Este guia fornece instruções detalhadas para a instalação e configuração do módulo de notificação via WhatsApp para WHMCS. O módulo permite o envio automático de mensagens de faturas, lembretes de vencimento e avisos gerais para clientes, utilizando a API do WhatsApp da Central Whats.

Requisitos

Antes de iniciar a instalação, certifique-se de atender aos seguintes requisitos:

  • WHMCS 8.0 ou superior
  • API pública de WhatsApp configurada
  • Acesso ao FTP do servidor
  • Permissões adequadas no WHMCS para instalar módulos
  • Conta ativa na Central Whats

Download e Upload dos Arquivos

Primeiro, você precisa fazer o download e upload dos arquivos do módulo:

Download: Baixe o arquivo ZIP do módulo usando o botão no topo da página.

Crie a estrutura de pastas no seu servidor:

/whmcs/modules/addons/whatsapp_notification/

Copie todos os arquivos extraídos para esta pasta:

  • api.php - Funções de comunicação com a API
  • hooks.php - Hooks do WHMCS para eventos automáticos
  • logs.php - Sistema de logs e relatórios
  • templates.php - Gerenciamento de templates
  • utils.php - Funções utilitárias
  • whatsapp_notification.php - Arquivo principal do módulo
  • whmcs.json - Metadados do módulo

Ativação do Módulo no WHMCS

Agora você precisa ativar o módulo no painel administrativo do WHMCS:

  1. Acesse o painel admin do WHMCS
  2. Vá em Configurações > Módulos Addon
  3. Localize o módulo "Notificação WhatsApp"
  4. Clique em "Ativar"
Importante: Certifique-se de que todos os arquivos foram enviados corretamente antes de tentar ativar o módulo.

Configuração da Central Whats

Para usar o módulo, você precisa de uma conta na Central Whats:

  1. Acesse centralwhats.pro
  2. Crie uma conta ou faça login
  3. No painel, vá em "Configurações" > "API"
  4. Copie o token de autenticação
  5. Conecte seu WhatsApp seguindo as instruções do painel
Dica: Mantenha o token em local seguro, pois ele será necessário para configurar o módulo.

Configuração do Módulo

Agora configure o módulo no WHMCS:

  1. No painel admin, vá em Addons > Notificação WhatsApp
  2. Na aba "Configurações", insira o token da Central Whats
  3. Configure as demais opções conforme necessário:
  • Token da API: Cole o token obtido da Central Whats
  • Formato da Data: Mantenha o padrão ou personalize
  • Enviar PDF da Fatura: Ative se desejar enviar PDFs automaticamente
  • Temporariamente Desativado: Use para pausar o envio de mensagens

Clique em "Testar Conexão" para verificar se tudo está funcionando.

Personalização de Templates

O módulo já vem com templates predefinidos, mas você pode personalizá-los:

Templates de Cliente:

  • Notificação de Fatura
  • Confirmação de Pagamento
  • Lembrete de Vencimento
  • Serviço Ativado/Suspenso
  • Boas-vindas

Templates de Admin:

  • Novo Pedido
  • Novo Ticket
  • Pagamento Recebido
Variáveis Disponíveis: Use variáveis como {firstname}, {amount}, {invoiceid} para personalizar as mensagens automaticamente.

Configurações Importantes

Para o funcionamento correto do módulo, observe estas configurações:

Formato do Telefone: O número de telefone dos clientes deve estar no formato internacional (Ex: 5511999999999)
WhatsApp Conectado: Certifique-se de que o WhatsApp esteja conectado no painel da Central Whats
PDFs de Fatura: Para enviar PDFs, ative a opção "Enviar PDF da Fatura" nas configurações
