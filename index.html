Tutorial de Instalação - Módulo WHMCS

Guia completo para instalar e configurar o módulo de notificação WhatsApp no seu WHMCS

ℹ️

Introdução

Este guia fornece instruções detalhadas para a instalação e configuração do módulo de notificação via WhatsApp para WHMCS. O módulo permite o envio automático de mensagens de faturas, lembretes de vencimento e avisos gerais para clientes, utilizando a API do Central Whats.

💡 Importante: Este módulo requer uma conta ativa no Central Whats e WHMCS versão 8.0 ou superior.
📋

1. Requisitos

Antes de iniciar a instalação, certifique-se de atender aos seguintes requisitos:

  • ✅ WHMCS 8.0 ou superior
  • ✅ Conta ativa no Central Whats
  • ✅ Acesso ao FTP do servidor
  • ✅ Permissões adequadas no WHMCS para instalar módulos
  • ✅ PHP 7.4 ou superior
📁

2. Download e Upload

Primeiro, você precisa criar a estrutura de pastas e fazer upload dos arquivos:

Estrutura de Pastas:

/whmcs/modules/addons/whatsapp_notification/

Arquivos do Módulo:

  • api.php
  • hooks.php
  • logs.php
  • templates.php
  • utils.php
  • whatsapp_notification.php
  • whmcs.json
⚠️ Atenção: Certifique-se de que todos os arquivos foram enviados corretamente e que as permissões estão adequadas (644 para arquivos).
⚙️

3. Ativação no WHMCS

Agora você precisa ativar o módulo no painel administrativo do WHMCS:

  1. Acesse o painel admin do WHMCS
  2. Vá em Configurações → Módulos Addon
  3. Localize "Notificação WhatsApp" na lista
  4. Clique em "Ativar"
  5. Configure as permissões de acesso conforme necessário
✅ Sucesso: Após a ativação, o módulo criará automaticamente as tabelas necessárias no banco de dados.
🔑

4. Configuração Central Whats

Para obter o token de API necessário:

  1. Acesse centralwhats.pro
  2. Crie uma conta ou faça login
  3. No painel, vá em "Configurações" → "API"
  4. Copie o Token de Autenticação
  5. Certifique-se de que seu WhatsApp está conectado
💡 Dica: Mantenha o token seguro e não o compartilhe. Ele é único para sua conta.
🔧

5. Configuração do Módulo

Configure o módulo no WHMCS:

  1. No WHMCS, vá em Addons → Notificação WhatsApp
  2. Na aba "Configurações", cole o token da API
  3. Configure o formato da data (padrão: %d.%m.%y)
  4. Ative/desative o envio de PDF das faturas
  5. Clique em "Salvar Configurações"
  6. Use o botão "Testar Conexão" para verificar
Token da API: [SEU_TOKEN_AQUI] Formato da Data: %d/%m/%Y Enviar PDF: ✅ Ativado Status: ✅ Ativo
📝

6. Personalização de Templates

O módulo vem com templates pré-configurados que você pode personalizar:

Templates Disponíveis:

  • 📋 Notificação de Fatura
  • 💰 Confirmação de Pagamento
  • ⚠️ Lembrete de Vencimento
  • 🚀 Serviço Ativado
  • 🔒 Serviço Suspenso
  • 👋 Boas-vindas
  • 🎫 Novo Ticket

Variáveis Disponíveis:

{firstname} - Nome do cliente {lastname} - Sobrenome do cliente {email} - Email do cliente {invoiceid} - Número da fatura {amount} - Valor da fatura {duedate} - Data de vencimento {service} - Nome do serviço {domain} - Domínio do serviço
⚡

7. Configurações Importantes

Para garantir o funcionamento perfeito:

📱 Formato do Telefone: O número de telefone dos clientes deve estar no formato internacional (Ex: 5511999999999)
🔗 Conexão WhatsApp: O WhatsApp precisa estar conectado no painel do centralwhats.pro
✅ Pronto! O módulo já está funcionando e enviará notificações automáticas conforme configurado.

  • ✅ Interface para envio manual de mensagens
  • ✅ Suporte a envio de PDF de faturas
  • ✅ Notificações para clientes e admins
  • ✅ Logs detalhados de todas as mensagens
